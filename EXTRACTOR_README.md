# ChatGPTDialogs Conversation Extractors

Набор инструментов для извлечения полных бесед из ChatGPT проекта.

## 📄 Скрипты

### 1. `complete_exporter.js` ⭐ **РЕКОМЕНДУЕМЫЙ**

**Назначение**: Полное извлечение всех сообщений из ChatGPT бесед

**Особенности**:
- ✅ Извлекает **ВСЕ сообщения** из разговора (не только активную ветку)
- ✅ Сортирует по времени создания
- ✅ Включает информацию о каждом сообщении (ID, время, роль)
- ✅ Автоматически скачивает JSON файлы

**Как использовать**:
1. Откройте нужную страницу ChatGPT в браузере
2. Откройте DevTools (F12 / Cmd+Option+I)
3. Перейдите в вкладку **Console**
4. Скопируйте весь код из этого файла
5. Вставьте в консоль и нажмите Enter
6. Подождите завершения (будут загружены все JSON файлы)

**Вывод**: Скачивает JSON файлы для каждой беседы в папку Downloads

---

### 2. `active_thread_exporter.js`

**Назначение**: Извлечение только активной ветки разговора

**Особенности**:
- Работает через ChatGPT API
- Извлекает текущую ветку (через `current_node`)
- ⚠️ **Ограничение**: Не захватывает альтернативные ветки разговора

**Статус**: Заменен на `complete_exporter.js`

---

### 3. `conversation_extractor.py`

**Назначение**: Обработка экспортированных JSON файлов с сервера ChatGPT

**Особенности**:
- Парсит формат ChatGPT API
- Восстанавливает активную ветку разговора
- Группирует сообщения по ролям (user/assistant)

**Входные данные**:
```
conversations.json  (полный экспорт от ChatGPT)
```

**Выходные данные**:
```
conversation_exports/
├── Dialog_Title_[id].json
├── Another_Dialog_Title_[id].json
└── ...
```

**Как использовать**:
```bash
python conversation_extractor.py
```

---

## 🔄 Рекомендуемый процесс

1. **Экспортирование**: Используйте `complete_exporter.js` в консоли браузера
2. **Сохранение**: Все файлы автоматически загружаются в Downloads
3. **Организация**: Переместите в `/Users/username/Development/GitHub/ChatGPTDialogs/exports/`

## ⚠️ Важно

- Нужен **доступ в консоль браузера** (Chrome DevTools)
- Требуется **авторизация в ChatGPT** (для получения access token)
- Скрипт не удаляет и не модифицирует оригинальные беседы
- JSON файлы содержат **полную историю** каждого разговора
